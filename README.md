# HomeLab

This repository documents my home lab infrastructure projects, covering enterprise networking,server role deployments, and cloud and identity integration. It servers as both a technical reference and a showcase of my hands on work as an IT professional.

## Equipment

Most of the home lab work I do is done on a custom-built desktop (ASUS motherboard) with an AMD Ryzen 5 3600 (6-core / 12-thread) CPU and 16 GB of DDR4-3600 memory. I run VMware Workstation Pro on this host; For anyone looking to build a similar environment on a budget, any modern desktop with a 6-core CPU and 16 GB of RAM is more than enough to follow along — a refurbished business desktop (Dell, HP, Lenovo) can be picked up second-hand for around £150–£300.
